## Goals (from README)

- Multi-token and stablecoin support (registry-driven token IDs).
- Multiple Ergo wallets (connect flow in the SDK).
- Referral validation with anti-abuse rules (API + policy).
- Analytics summary and campaign views.
- Dockerized stack with environment-based configuration (no hardcoded RPC URLs).
